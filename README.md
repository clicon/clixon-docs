# clixon-docs
Clixon documentation.

build:
````bash
$ apt install python3
$ python3 -m venv env
$ source env/bin/activate
$ pip install -r requirements.txt
$ make html
````

See [Documentation project status](https://readthedocs.org/projects/clixon-docs/)
